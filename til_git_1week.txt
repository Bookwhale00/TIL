2. 1주차 주요 개념 키워드 적어보기

git = 버전관리 시스템

로컬저장소와 원격저장소

로컬repo는 내 컴퓨터, 원격repo는 Github

1) 원격에 있고 로컬에 없을 때는 Clone해서 연결
2) 로컬에 있고 원격에 있을 때 원격repo를 만든 뒤 
tracking(추적)해서 연결

로컬-원격 연결할 때 이름은 origin, branch는 main

파일을 변경하고 ->add(스테이징) -> commit -> push

원격에 있는 파일이 변경되면 pull -> (add) commit -> push
